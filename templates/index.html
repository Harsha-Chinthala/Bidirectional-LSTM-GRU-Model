Introduction

This project focuses on developing a chatbot system for sentiment analysis using deep learning models, analyzing customer reviews to classify them as positive or negative or neutral, and providing real-time insights for businesses. The project employs a dataset of Amazon reviews, preprocessed with techniques like stopwords removal and Word2Vec, using Bidirectional GRU and LSTM layers for context capture, trained on one million review samples, and evaluated for various performance metrics. The chatbot interface allows user interaction for real-time sentiment analysis.
